Design Principles

Whitespace Visual Hierachy Contrast

Whitespace

Apple uses generous amounts of whitespace. This works together with the thin typeface and large font sizes to make sure that there isn’t too much information on the screen at once. Apple also places imagery strategically to draw the users eye to specific elements down the page.

Visual Hierachy

Simply Chocolate, is a great example of how to use size and scale to emphasize visual hierarchy. The first thing you’re drawn to in the design is the huge text that explains what the product is all about. This is done in a way that doesn’t detract from the other elements on the screen.

Contrast

Just as color can be applied to typographic headings to create contrast, so can size. The website of Dropbox, uses big headings to catch the visitor’s attention and draw them in so that they read more of the smaller paragraphs below.
